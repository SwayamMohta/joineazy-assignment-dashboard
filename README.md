# JoinEazy Role-Based Dashboard

### Frontend Intern Round 2 Evolution

---

## Project Overview

JoinEazy is a high-performance academic management dashboard designed for **Professors** and **Students**. This Round 2 evolution introduces deep role-based workflows, a collaborative group management engine, and real-time submission tracking. 

The application features a premium "human-coded" aesthetic, full dark mode support, and robust data persistence via Local Storage.

Dashboard : https://joineazy-assignment-dashboard.vercel.app

---

## Key Features

### Professor Workflow
- **Academic Ledger**: Manage all course assignments and deadlines.
- **Submission Tracking**: Real-time visibility into student and group progress.
- **Professional Reviews**: Evaluate submissions with performance grades and actionable feedback.
- **Class Analytics**: High-level cohort metrics and completion rates.

### Student Workflow
- **Academic Pipeline**: Track personal work backlog and milestones.
- **Collaboration Hub**: Form or join student groups for collaborative tasks.
- **Submission Protocol**: Integrated acknowledgment system with "Leader-only" finalization for groups.
- **Learning Momentum**: Visual progress tracking across all enrolled courses.

### Common Infrastructure
- **Mock Identity System**: Seamlessly switch between Professor and Student roles.
- **Responsive Navigation**: Full mobile support with a dedicated sidebar toggle.
- **Premium UI**: Linear-inspired dark/light theme with Framer Motion transitions.
- **Persistence**: All user states, group memberships, and assignments persist across sessions.

---

## Tech Stack

- **React.js**: Functional components with hooks.
- **Vite**: Ultra-fast build tool.
- **Tailwind CSS**: Utility-first styling with custom design tokens.
- **Framer Motion**: Smooth page transitions and micro-animations.
- **Material Symbols**: Integrated iconography.
- **Local Storage**: Reliable data persistence without backend overhead.

---

## How to Run

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Start development server**:
    ```bash
    npm run dev
    ```
3.  **Access the dashboard**: Open `http://localhost:5173`.
